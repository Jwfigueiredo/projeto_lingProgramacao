🧾 Nome do Projeto

Sistema de Consulta de FIIs

📌 Descrição

Aplicação web que permite consultar Fundos Imobiliários em tempo real utilizando API externa, exibindo dados em uma tabela dinâmica com controle de ativos.

🚀 Funcionalidades
Consulta de FIIs via API
Tabela dinâmica
Remoção individual de ativos
Controle de duplicidade
Limite de ativos
Botões de seleção rápida
Limpeza total da tabela


🛠️ Tecnologias
HTML5
CSS3
JavaScript (ES6 Modules)
API REST (brapi.dev)
